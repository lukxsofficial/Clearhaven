# Clearhaven
Clearhaven is a lightweight Vanilla+ modlist for Skyrim AE built around stability, visual clarity, and performance.
It does not aim to reinvent Skyrim — it refines it.
With around 250 carefully selected mods, Clearhaven enhances the base game while preserving its original identity. The goal is simple: Skyrim as you remember it — cleaner, smoother, and more consistent.

# Philosophy

Clearhaven focuses on:
Visual improvements without overhauling Skyrim’s identity
First-person friendly gameplay
Stable performance on modern mid-range systems
Minimal scripting and reduced complexity

There is no DynDOLOD setup, no Grass Cache generation, and no seasonal overhauls. Installation is straightforward and maintenance is minimal.

# Performance Targets

Clearhaven is designed for modern mid-range hardware (2021+).

Recommended for 1080p:

- Ryzen 5 5600 or equivalent
- RTX 3060 / RX 6600 or better
- 8GB VRAM minimum
- 16GB RAM minimum

Recommended for 1440p:

- Ryzen 5 5600 or better
- RTX 3070 / RX 6700 XT or better
- 8–12GB VRAM
- 32Gb RAM optimal

Actual performance may vary depending on system configuration.

# Installation

Clearhaven is distributed via Wabbajack.

- Install a clean Skyrim Anniversary Edition.

- Use Wabbajack to install Clearhaven.

- Follow the included post-install instructions.

No manual DynDOLOD generation or Grass Cache setup required.
But if you know what you are doing, you can generate it.

# Maintenance & Updates

Clearhaven is a side project maintained alongside my primary modlist, Ashveil.
Updates will be released when meaningful improvements are ready. There is no fixed update schedule.

# Support

Clearhaven is provided as-is.

Before reporting issues:
- Ensure you followed installation instructions exactly.
- Do not add or remove mods unless you understand the implications.

Modified installations are not officially supported.
